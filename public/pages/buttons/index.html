Buttons

  • Pandora 2.0
  • UI
  • Buttons
Default Buttons

Use the button classes on an <a>, <button>, <input>, or <span> element. 


            <button class="button">Default</button>
Rounded Buttons

Add .pill-button to a button to make it rounded. 


            <button class="button pill-button">Default</button>
Outline Buttons

Use a class .outline to create outline buttons. 


            <button class="button outline">Default</button>
Outline Rounded Buttons

Add .pill-button to a button to make it rounded. 


            <button class="button pill-button outline">Default</button>
Soft Buttons
New

Use a class .soft-*** to create soft buttons where *** is the color name. 


            <button class="button soft-sand">Sand</button>
Icon Buttons

You can use any icon fonts or image to add icon into button. You can create these buttons with or without a caption.


            <button class="button">
                <span class="icon mif-checkmark"></span>
                <span class="caption">Success</span>
            </button>

            <button class="button">
                <img class="icon" src="/images/git.svg" alt="Git"/>
                <span class="caption">Git</span>
            </button>
Button Sizes

Add classes large, small, or mini to change button size.


            <button class="button large">Large</button>
            <button class="button small">Small</button>
            <button class="button mini">Mini</button>
Button Types

Add classes .square, or .cycle to change a button type.


            <button class="button square">
                <span class="icon mif-github"></span>
            </button>

            <button class="button cycle">
                <span class="icon mif-github"></span>
            </button>
Button with Animated Border
New

Add z-index to button container (you can use classes .z-{1..10}), and then add classes .animated-border, or .animated-border-hover to set animated border for button. 


            <button class="button animated-border">
                <span class="icon mif-github"></span>
                <span class="caption">GitHub</span>
            </button>

            <button class="button animated-border-hover">
                <span class="icon mif-github"></span>
                <span class="caption">GitHub</span>
            </button>

            <button class="button animated-border-blue">
                <span class="icon mif-github"></span>
                <span class="caption">GitHub</span>
            </button>
