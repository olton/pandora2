About Grid System

You can create a grid with built-in grid system. The grid system uses a series of containers, rows, and columns to layout and align content. It’s built with flexbox and is fully responsive. The following css classes are available for building a grid: .grid, .row, .cell-*, .cell-*-*, .cell-*-half, .cell-*-one-third, .cell-*-two-third, .cell-*-one-fifth, .cell-*-two-fifth, .cell-*-three-fifth, .cell-*-four-fifth, .offset-*, and .offset-*-*.

Grid Gutter Size

Grid System used --grid-gutter variable to set gutter width. To set gutter width to 10px, you can use the following code:

Global setup:

        :root {
            --grid-gutter: 10px;
        }
Local setup:

        <row style="--grid-gutter: 10px;">
            ...
        </row>
Grid Code

        <div class="grid">
            <div class="row">
                <div class="cell-md-1"></div>
                <div class="cell-md-3"></div>
                <div class="cell-md-8"></div>
            </div>
        </div>
