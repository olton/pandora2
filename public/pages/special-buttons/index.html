Breadcrumbs

You can create breadcrumbs with .breadcrumbs class anb the specifier .arrow or .default to change the style of breadcrumbs.

Default
  • Home
  • Library
  • Data
  • Bootstrap

            <ul class="breadcrumbs default">
                <li><a href="#"><span class="mif-home icon"></span>Home</a></li>
                <li><a href="#">Library</a></li>
                <li><a href="#">Data</a></li>
                <li class="active"><a href="#">Bootstrap</a></li>
            </ul>
Arrow
  • Home
  • Library
  • Data
  • Bootstrap

            <ul class="breadcrumbs arrow">
                <li><a href="#"><span class="mif-home icon"></span>Home</a></li>
                <li><a href="#">Library</a></li>
                <li><a href="#">Data</a></li>
                <li class="active"><a href="#">Bootstrap</a></li>
            </ul>
