Welcome to Pandora 2.0

Pandora - is a set of template for building administrative websites. This set created with Metro UI. The template contains different types of reusable and professional responsive components with different variations.

This set is built using only the Metro UI component library, excluding ApexCharts, a great tool for creating various graphs and charts. All charts in Pandora were created with ApexCharts.

Set includes:

Common
Brief description and demonstration of common styles and javascript utilities and libraries.
UI Elements
Brief description and demonstration of Metro UI components and widgets.
Demo Pages
Ready to use demo pages.
